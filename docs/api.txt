登录界面：
	请求方式：post
	请求方式：{
		name: 用户名,
		password: 用户密码
	}
	返回参数： {
			errCode: 0  正常
					 1  密码错误
			msg: 错误信息
	}


注册界面：
	请求方式：post
	请求参数：{
		name: 用户名,
		password: 用户密码,
		email: 用户邮箱,
		phone: 手机号码（可选）
	}
	返回参数：{
		errCode: 0  正常
				 1  密码错误
				    邮箱格式不对
				    邮箱已被注册
		msg: 错误信息
	}

个人转让：
	请求方式：post
	请求参数：{
		good_name: 商品名称
		good_info: 商品信息
		type: 个人转让或公益转手（0现金or1积分）
		good_price: 商品价格
		good_integral: 商品积分
		good_image: 图片路径（图片名字）
		contacts: 联系方式（联系地址）
		remark: 备注信息
		good_storage: 商品库存（页面提示为：商品数量）
		user_id: 商品所属用户的id（考虑从后端插入）
		class_id: 商品所属分类的id（考虑从后端插入）
	}
