登录界面：
	请求方式：post
	请求方式：{
		name: 用户名,
		password: 用户密码
	}
	返回参数： {
			errCode: 0  正常
					 1  密码错误
			msg: 错误信息
	}


注册界面：
	请求方式：post
	请求参数：{
		name: 用户名,
		password: 用户密码,
		email: 用户邮箱,
		phone: 手机号码（可选）
	}
	返回参数：{
		errCode: 0  正常
				 1  密码错误
				    邮箱格式不对
				    邮箱已被注册
		msg: 错误信息
	}
